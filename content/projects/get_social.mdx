Get_Social is a Social Media Website built using the MERN (MongoDB, Express.js, React.js, Node.js) stack is a comprehensive platform that allows users to connect, share, and interact with each other. It offers a range of features and functionalities typically found in popular social media websites.

## Description

- Technology Stack: The website is built using the MERN stack, consisting of MongoDB for the database, Express.js for the backend framework, React.js for the frontend, and Node.js for the server-side runtime environment.

- User Registration and Authentication: Users can create accounts, log in securely, and maintain personal profiles.

- Content Sharing: Users can create posts, share text, images, or videos, and interact with posts through comments, likes, and shares.

- User Interactions: Users can follow other users, receive notifications for new followers.

- User Profiles: Each user has a profile page displaying their bio, posts, followers, and following.

- Real-Time Updates: The website provides real-time updates on new posts, comments, or likes.

- Responsive Design: The website is built with a responsive design, ensuring compatibility and optimal user experience across different devices.
